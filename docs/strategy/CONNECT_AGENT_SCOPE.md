# Escopo comercial — Shamar Connect e Shamar Agent

Este documento orienta site, propostas, copy, roadmap e prompts futuros.

## Posicionamento

O Shamar Connect é uma central de atendimento, relacionamento e vendas.

O Shamar Agent é um conector local de integração.

## Promessa principal

Organizar o atendimento para que a empresa pare de perder clientes por falta de processo.

## O que podemos prometer

- centralizar conversas
- manter histórico do cliente
- organizar responsáveis
- separar setores e filas
- registrar notas internas
- usar respostas rápidas
- acompanhar retornos
- apoiar pré-venda e pós-venda
- consultar dados autorizados do sistema atual via Agent

## O que não podemos prometer

- trocar o ERP do cliente
- substituir sistema fiscal
- substituir estoque completo
- substituir financeiro interno
- fazer prontuário médico
- automatizar conversa sem humano
- vender o Agent como IA
- vender chatbot como se fosse Agent

## Frases aprovadas

`Central de atendimento, relacionamento e vendas.`

`Humano no controle, processo visível.`

`O Shamar Connect organiza o atendimento. O Shamar Agent conecta informações autorizadas do sistema atual.`

`O sistema atual continua cuidando da gestão interna.`

## Frases proibidas ou arriscadas

`ERP completo.`

`Substitui seu sistema atual.`

`Robô de atendimento automático.`

`IA que atende seus clientes sozinha.`

`Estoque completo dentro do Shamar.`

`Sistema fiscal integrado completo.`

## Regra de ouro

Sempre explicar que o Connect organiza a entrada, o relacionamento e o acompanhamento do cliente.

Sempre explicar que o Agent é ponte técnica com o sistema existente.
